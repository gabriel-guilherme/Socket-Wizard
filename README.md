### Pré-requisitos: Node.js

### Passos para Execução:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/gabriel-guilherme/Socket-Wizard.git
   cd Socket-Wizard
   ```

2. **Inicie o servidor:**
   ```bash
   node server.js
   ```
   
3. **Abra o seu navegador e acesse::**
   ```bash
   (http://localhost:3000)
   ```
